## Instructions


You can use this commands after selecting table type and player type

Load game from a file
LOAD <filename.txt>

Save game to a file
SAVE <filename.txt>

## Save info explanation

Save system works like this.

Save System Architecture
-- Shape Type -- (int)
-- Player Type -- (int)
-- Move Count -- (int)
-- Game Board -- (GAME ENUM)
